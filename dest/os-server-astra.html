О компонентах
Единый дистрибутив Astra Linux содержит полный набор компонентов, необходимых для инсталляции на серверы и управления. Чтобы ими воспользоваться, достаточно при установке ОС выбрать серверную модель в качестве основной.
Подробнее

Ключевые улучшения:

Для различных процессорных архитектур разработаны специальные варианты исполнения с защитой всех компонентов и программных ресурсов

Способ лицензирования
Необходима отдельная лицензия на каждый одно — или двухсокетный сервер и на каждую виртуальную машину. Если сервер четырехсокетный, потребуется 2 лицензии.

Astra Automation

Платформа для автоматизированного создания и сопровождения ИТ-инфраструктуры на базе продуктов
ГК «Астра». Позволяет развернуть серверную ОС с набором необходимого софта ГК «Астра» из готовых
шаблонов максимально просто и быстро: буквально в несколько кликов и без глубоких знаний в области Linux.

  • Для всех поддерживаемых обновлений Astra Linux
