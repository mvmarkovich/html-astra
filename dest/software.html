Ready For Astra

Полный программный стек для решения максимально широкого круга практических задач

Наша цель — дать клиентам наиболее полный набор ИТ-инструментов для решения бизнес-задач в организациях любого масштаба: от микропредприятий до крупных корпораций с разветвлённой филиальной сетью. Всё прикладное ПО ГК «Астра» является российским и внесено в национальный Реестр Министерства цифрового развития.

Операционная система Astra Linux

Не имеющая аналогов сертифицированная операционная система со встроенными средствами защиты информации для стабильных и безопасных ИТ-инфраструктур любого масштаба и работы с данными любой степени конфиденциальности.

Брест

Программный комплекс «Средства виртуализации „Брест“ — российская платформа со встроенными средствами защиты информации ОС Astra Linux Special Edition для создания и управления виртуальными ИТ-инфраструктурами любой сложности.
Узнать больше

Termidesk

Инфраструктура виртуальных рабочих мест (ВРМ) Termidesk — мультиплатформенный продукт для организации безопасной и удалённой работы сотрудников. Termidesk устойчив к внешнеполитическим рискам — центр разработки и репозиторий находятся в России и не подвержены иностранному влиянию.

RuBackup

Профессиональное программное решение резервного копирования, ориентированное на ИТ-инфраструктуру государственных и корпоративных заказчиков, которые переходят на российское программное обеспечение
Узнать больше

ALD Pro

Программный комплекс на базе ОС Astra Linux для централизованного управления объектами организаций различного масштаба
Узнать больше

WorksPad

WorksPad — клиент-серверное корпоративное решение класса Enterprise Mobility Management (EMM) для организации защищённой мобильной работы сотрудников на смартфонах и планшетах под управлением iOS и Android.
Узнать больше

RuPost

Система управления корпоративной почтой, легко интегрируется в ИТ-инфраструктуру на основе OC Astra Linux. Поддерживает кластеризацию и может масштабироваться без ограничений. Полностью совместима со всем стеком ПО производства ГК «Астра».
